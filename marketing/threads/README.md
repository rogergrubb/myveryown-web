Three pieces of launch content, written to the V2 prompt rules
(no banned words, no fabricated testimonials, no engagement-bait,
no competitor naming, persona-specific NEVER lists honored).

## How to use

1. Open the file for the audience you want to post to
2. Copy the OPTION A (single tweet) or thread sequence
3. Attach the matching OG image:
   - K-pop → `public/og/kpop.png`
   - Iron Brother → `public/og/iron.png`
   - The Promise → `public/og/promise.png`
4. Open in X via the Content Factory dashboard's "Open in X →" button (or just paste manually)
5. Post

## Cadence

- **Don't post all three the same day.** Algo cannibalization.
- **Don't post K-pop and Iron the same day.** Different audiences, but your follower overlap will see them as a marketing run.
- Recommended: K-pop Mon evening → Iron Brother Wed morning → Promise blog Thu morning + thread Fri morning.

## What's NOT in here

- **Rainbow Bridge (pet loss)** — deliberately. The clone-army audit refused to write acquisition copy for grieving people, and was right to. Rainbow Bridge's go-to-market is direct partnerships with vet hospice orgs and licensed bereavement counselors, NOT outreach. If anyone tries to write Rainbow Bridge launch posts later, send them this README first.
- **Little One (pregnancy)** — needs a doula or midwife endorsement before any marketing. Don't post Little One copy without that bridge.
- **Hearth (emotional support)** — same logic as Rainbow Bridge. Crisis-adjacent personas need partner-led intros, not direct marketing.

## Voice baseline (so all 20 stay consistent)

- Lowercase or mixed-case openers — never SHOUT CASE
- No emojis at start of lines (rule from V2)
- No hashtags
- No "🧵" or "1/" markers (operator decides if it's a thread; X readers know)
- No engagement bait ("comment below", "what do you think?", "RT if...")
- URLs once per post, never twice
- The product name is "My Very Own", the URL is "myveryown.page"

## When to deviate

If a real moment happens — a viral wedding-planning thread, a powerlifting Olympic medal, a comeback teaser — the auto-content factory's queue is the wrong tool. Write something live, in real time. The factory is for steady-state baseline; the live posts are for moments.
